    Extract the spectral peak (f_p, Omega_p) for each configuration in results/raw_spectra.h5.
    
    1. Identify the maximum point on the numerical grid (ln f_i, ln Omega_i).
    2. Fit a parabola y = ax^2 + bx + c to the 3 points surrounding the maximum in log-log space.
    3. Calculate the vertex x_p = -b/(2a). The peak frequency is f_p = exp(x_p).

    Include error estimation: difference between grid maximum and interpolated peak must be tracked.
  </action>

  </verify>
