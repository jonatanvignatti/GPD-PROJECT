<task type="auto">
  <name>Task 1: Compute Sensitivity Jacobian using Finite Differences</name>
  <files>results/sensitivity_matrix.json, analysis/sensitivity_analysis.py</files>
  <action>
    Compute the Jacobian J = partial(ln f_p, ln Omega_p) / partial(ln v_w, ln alpha) from the data in results/peak_trajectories.csv.
    
    1. Organize peak positions into a grid of (v_w, alpha).
    2. Use central finite differences for internal grid points: dY/dX = (Y_{i+1} - Y_{i-1}) / (X_{i+1} - X_{i-1}).
    3. Use forward/backward differences for boundary points.
    4. Compute the Jacobian for both SSM and Envelope models separately.
    5. Evaluate the "Signal Drift" vector magnitude |D| = sqrt((delta ln f_p)^2 + (delta ln Omega_p)^2).
    
    Save coefficients and error estimates to JSON.
  </action>
  <verify>
    1. Dimensionality: Jacobian components must be dimensionless.
    2. Consistency: J_vw_Op for bubbles should be positive (Omega_p increases with v_w).
    3. Error Budget: Check if central vs forward differences agree within 20% in the linear regime.
  </verify>
  <done>Sensitivity matrix computed and saved to results/sensitivity_matrix.json</done>
</task>
